### 如何自動化清理

定義標準化流程：</br>
輸入原始檔 -> 自動檢查問題 -> 執行清洗邏輯 -> 輸出報表</br>

實作邏輯
1. 輸入 (Input)：設定一個資料夾路徑，程式自動抓取該路徑下所有的 .csv 檔。
2. 自動檢查 (Check)：
    - df.isnull().sum()：自動統計每一欄有多少空值。
    - df.dtypes：檢查資料型態是否符合預期（如：金額應為數字而非字串）。
3. 執行清洗 (Clean)：呼叫預先寫好的清洗函數。
4. 輸出報表 (Output)：使用 to_csv() 匯出清洗後的檔案，並用 print() 輸出簡單的清洗日誌（Log）。

### AI 協作思維：運用 Gemini 生成邏輯
試著下Prompt讓 Gemini 幫忙寫出複雜的判斷式。</br>

操作場景：處理複雜的「金額」欄位</br>
假設資料中的金額長這樣："$ 25,000" 或 "25000元"，這無法直接計算。</br>
```
import pandas as pd

# 模擬包含各種「髒金額」的資料
data = {
    'OrderID': [101, 102, 103, 104, 105, 106],
    'Product': ['Laptop', 'Mouse', 'Keyboard', 'Monitor', 'Tablet', 'Webcam'],
    'Amount': ['$ 25,000', '800元', '$ 1,200', '5000', '12,000元', '未知'] # 包含各種雜訊
}

df = pd.DataFrame(data)

print("--- 清理前的原始資料 ---")
print(df)
print("\nAmount 欄位的資料型態:", df['Amount'].dtype)
```
對 Gemini 輸入以下 Prompt：</br>

「我有一欄 Pandas DataFrame 的資料叫 'Amount'，內容包含錢字號、逗號和中文字，例如 '$ 1,200' 或 '500元'。請幫我寫一個 Python 函數，可以自動把這些符號去掉，並將該欄位轉換為整數 (int)，如果遇到無法轉換的內容就填入 0。」</br>

Gemini 會產出類似下方的代碼：</br>
```
def clean_currency(value):
    if isinstance(value, str):
        # 移除 $, 逗號, 和 "元"
        clean_str = value.replace('$', '').replace(',', '').replace('元', '').strip()
        return int(clean_str) if clean_str.isdigit() else 0
    return value

# 自動化套用
df['Amount'] = df['Amount'].apply(clean_currency)
```
執行提供的 clean_currency 函數。透過 apply() 方法，可以將複雜的字串處理邏輯一次性套用到整欄資料。
```
def clean_currency(value):
    if isinstance(value, str):
        # 移除 $, 逗號, 和 "元"
        # 提示學員：這部分邏輯就是透過 Gemini 描述需求後生成的
        clean_str = value.replace('$', '').replace(',', '').replace('元', '').strip()
        
        # 判斷剩餘字串是否為數字，若不是（例如「未知」）則轉為 0
        return int(clean_str) if clean_str.isdigit() else 0
    return value

# 執行清理
df['Amount'] = df['Amount'].apply(clean_currency)

print("--- 清理後的資料 ---")
print(df)
print("\nAmount 欄位的資料型態:", df['Amount'].dtype)
```
驗證與進階計算
清理後的 Amount 已經變成 int64（整數）型態，這時候就可以進行數學運算了。
```
# 驗證是否可以計算總和
total_sales = df['Amount'].sum()
print(f"清理成功！總銷售金額為：{total_sales}")
```
