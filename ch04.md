# 函式 def
函式就像是一個「自動販賣機」或「黑盒子」。你給它一些輸入（投幣/選擇），它會幫你執行特定的任務，最後可能給你一個結果（飲料）。</br>
使用函式的優點：</br>
- 程式碼重用：寫一次，用多次。
- 易於維護：如果要修改邏輯，只需要改一個地方。
- 結構清晰：把複雜的問題拆解成多個小任務。

## 基本語法
在 Python 中，我們使用 def 關鍵字來定義函式：
```
def 函式名稱(參數1, 參數2):
    # 函式內部的程式碼塊
    實作內容
    return 回傳值
```

## 無參數、無回傳
```
def say_hello():
    print("你好！歡迎學習 Python。")
    print("這是你的第一個函式。")

# 呼叫函式
say_hello()
```

## 帶有參數的函式
```
def calculate_area(width, height):
    area = width * height
    print("長度為", width)
    print("寬度為", height)
    print("計算出的面積為", area)

# 呼叫函式並傳入不同的數值
calculate_area(10, 5)
calculate_area(7, 3)
```

## 帶有回傳值 (Return)
```
def add_numbers(a, b):
    result = a + b
    return result

# 取得回傳值並存入變數
total = add_numbers(100, 250)

# 現在我們可以使用這個 total 變數做其他事
print("總和是", total)
print("總和的兩倍是", total * 2)
```

## 預設參數值 (Default Arguments)
可以為參數設定「預設值」，如果呼叫時沒有提供該參數，就會使用預設的內容。
```
def greet(name, message="你好"):
    print(message, name)

# 情況 1：提供所有參數
greet("YTLee", "早安")

# 情況 2：只提供 name，message 使用預設值
greet("小明")
```

### 如何自動化清理

定義標準化流程：</br>
輸入原始檔 -> 自動檢查問題 -> 執行清洗邏輯 -> 輸出報表</br>

實作邏輯
1. 輸入 (Input)：設定一個資料夾路徑，程式自動抓取該路徑下所有的 .csv 檔。
2. 自動檢查 (Check)：
    - df.isnull().sum()：自動統計每一欄有多少空值。
    - df.dtypes：檢查資料型態是否符合預期（如：金額應為數字而非字串）。
3. 執行清洗 (Clean)：呼叫預先寫好的清洗函數。
4. 輸出報表 (Output)：使用 to_csv() 匯出清洗後的檔案，並用 print() 輸出簡單的清洗日誌（Log）。

提示詞：</br>

有一個 Pandas 的 DataFrame，其中『日期』欄位的格式很亂，包含 2024-01-01、2024/01/02 和 2024.01.03。寫一個 Python 函數，將這些不同的格式統一轉換為標準的 YYYY-MM-DD 日期格式，若遇到無法轉換的內容請設為空值。</br>
```
import pandas as pd

def clean_date_format(input_df, column_name='日期'):
    """
    自動化日期清洗機器人
    1. 統一將分隔符 (. /) 轉換為標準的 (-)
    2. 強制轉換為標準 Datetime 物件
    3. 報表化處理結果
    """
    df = input_df.copy()

    # 確保欄位存在
    if column_name in df.columns:
        # 轉為字串並統一分隔符號
        df[column_name] = df[column_name].astype(str).str.replace('.', '-', regex=False)
        df[column_name] = df[column_name].str.replace('/', '-', regex=False)

        # 轉換為日期型別，無法解析的會變成 NaT (空日期)
        df[column_name] = pd.to_datetime(df[column_name], errors='coerce')

        # 輸出處理日誌
        nat_count = df[column_name].isna().sum()
        print(f"✅ '{column_name}' 欄位處理完成！")
        if nat_count > 0:
            print(f"⚠️ 注意：有 {nat_count} 筆資料因格式過於混亂無法解析，已轉為 NaT。")

    return df
```

有一份銷售資料的 DataFrame，其中的『銷售額』欄位包含了部分負數（例如 -500），這是不合理的。寫一個 Python 函式，自動將所有負數轉為正數（絕對值），並在處理完後告訴我一共修正了多少筆異常數值。</br>
```
def fix_numeric_logic(input_df, column_name='銷售額'):
    """
    數值邏輯修復機器人
    1. 偵測指定欄位中的負數
    2. 使用 .abs() 將其轉為正數
    3. 回報修正筆數
    """
    df = input_df.copy()

    if column_name in df.columns:
        # 偵測負數的數量（用於日誌紀錄）
        negative_count = (df[column_name] < 0).sum()

        # 執行絕對值轉換
        df[column_name] = df[column_name].abs()

        print(f"✅ '{column_name}' 欄位處理完成！")
        print(f"修正紀錄：共偵測到 {negative_count} 筆負數異常值，已自動轉為正值。")
    else:
        print(f"❌ 找不到欄位：{column_name}")

    return df
```

『客戶姓名』欄位有很多不必要的空格（例如：『李 華』），請幫我全部去除。</br>
『電話號碼』格式很亂，包含橫線、空格（如：0912-345-678 或 0911 222 333），寫一個 Python 函式，將這些符號全部移除，統一轉換成純數字字串。 請將這些邏輯包裝成一個名為 clean_strings 的函式。
```
def clean_string_data(input_df):
    """
    字串標準化機器人
    1. 移除姓名中的所有空格
    2. 移除電話中的橫線 '-' 與 空格 ' '
    """
    df = input_df.copy()

    # 清理客戶姓名：去除所有空格
    if '客戶姓名' in df.columns:
        df['客戶姓名'] = df['客戶姓名'].astype(str).str.replace(' ', '', regex=False)

    # 清理電話號碼：移除橫線與空格
    if '電話號碼' in df.columns:
        # 使用正則表達式 [-\s] 代表橫線或任何空白字元
        df['電話號碼'] = df['電話號碼'].astype(str).str.replace(r'[- ]', '', regex=True)

    print("✅ 字串標準化處理完成！（已修正姓名空格與電話格式）")
```

    # 印出完整日誌供檢查
    print("\n".join(log_details))

    # 將日誌存成文字檔備查
