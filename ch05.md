# 動態圖表與 Plotly 互動式設計

靜態 vs 互動：

- 靜態圖 (Matplotlib/Seaborn)：繪圖引擎必須在伺服器端（Colab 的 Linux 電腦）找到字體，把字「畫」進圖片裡。

- 互動圖 (Plotly)：繪圖引擎是你的「瀏覽器」。只要你的 Windows 或 Mac 裡有中文字體，Plotly 就能顯示中文。

## 靜態圖表
視覺化（Data Visualization）是溝通數據洞察最直觀的工具。雖然現代技術追求互動與動態效果，但「圖表」依然是學術論文、專業報告與商業提案中的核心標準。以下是 Python 常使用的視覺化模組：Matplotlib 與 Seaborn

1. Matplotlib：靈活的畫筆
Matplotlib 是 Python 視覺化界的祖師爺。它採用物件導向的邏輯，將圖表拆解為「畫布（Figure）」與「座標系（Axes）」。

    - Figure：就像一張空白的紙。
    - Axes：紙上的繪圖區域，包含座標軸、標題、數據點。

2. Seaborn：統計之美
Seaborn 是基於 Matplotlib 開發的高階繪圖庫，它與 Pandas 資料框架（DataFrame）高度整合。。它的目標是「讓困難的圖表變得簡單」，特別擅長處理 Pandas DataFrame 並自動計算統計指標（如誤差區間、分佈狀態）。
    - 預設美學設計：內建多種精美的主題（Styles）與調色盤（Color Palettes），比起原生的 Matplotlib，Seaborn 畫出來的圖表更具現代感。
    - 統計自動化：能自動執行複雜的統計運算（如計算平均值、標準差、回歸線、分佈密度等），並直接反映在圖表上。
    - 與 Pandas 完美對接：只需要告訴它 DataFrame 的欄位名稱，它會自動幫你處理索引、標籤與圖例。

### 多圖層銷售數據分析（Matplotlib）
```
#自定義標籤與繪圖細節。

import matplotlib.pyplot as plt
import pandas as pd

# 1. 準備數據
months = ['Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun']
sales_2024 = [120, 150, 180, 140, 210, 240]
sales_2025 = [135, 160, 200, 190, 230, 280]

# 2. 建立畫布與座標軸
fig, ax = plt.subplots(figsize=(10, 6))

# 3. 繪製兩條數據線
ax.plot(months, sales_2024, label='2024 銷售', marker='s', color='#A9A9A9', linestyle='--')
ax.plot(months, sales_2025, label='2025 銷售 (目標)', marker='o', color='#1f77b4', linewidth=3)

# 4. 靜態圖表特有的細節控制：標註最大值
max_val = max(sales_2025)
ax.annotate(f'銷售尖峰: {max_val}', xy=('Jun', max_val), xytext=('May', max_val+20),
             arrowprops=dict(facecolor='black', shrink=0.05))

# 5. 優化外觀
ax.set_title('2024-2025 上半年銷售對比分析', fontsize=16)
ax.set_xlabel('月份')
ax.set_ylabel('銷售金額 (萬美元)')
ax.legend()
ax.grid(axis='y', alpha=0.3)

plt.show()

print("說明：靜態圖表雖然不可縮放，但透過精確的箭頭標註（Annotate），能直接引導讀者閱讀重點。")
```
### 多維度分佈分析（Seaborn）
```
import seaborn as sns
import matplotlib.pyplot as plt

# 載入數據
penguins = sns.load_dataset("penguins")

plt.figure(figsize=(10, 6))

# 繪製圖表
sns.boxplot(data=penguins, x="species", y="body_mass_g", hue="sex", palette="Set2")

# 檢查中文是否正常
plt.title("不同性別與物種企鵝的體重分佈箱形圖")
plt.xlabel("物種")
plt.ylabel("體重 (克)")

plt.show()
```
