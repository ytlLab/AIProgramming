# Google Colab 簡介

Google Colaboratory（簡稱 Colab）是一款基於雲端的 Jupyter Notebook 環境。它最大的特色是完全免費且無需安裝，只要有 Google 帳號與瀏覽器即可撰寫並執行 Python 程式碼。

## 核心優點
- 免費 GPU/TPU 資源：這對機器學習與深度學習初學者極具吸引力，能大幅縮短運算時間。
- 雲端整合：程式碼自動儲存於 Google Drive，方便隨時切換裝置或與他人協作（類似 Google 文件的共享機制）。
- 預載環境：內建了常用的大數據與 AI 套件（如 NumPy, Pandas, TensorFlow），省去設定環境的煩惱。

# pyton 基礎語法

## 基本輸出輸入與變數
程式的基礎在於資料的流動：獲取資料並呈現結果。
- 變數賦值：x = 10
- 輸出：使用 print()。
- 輸入：使用 input() (注意：輸入進來的資料預設都是「字串」)。

```
#訪客歡迎程式 (字串相加)
#範例：
last_name = input("請輸入姓氏：")
first_name = input("請輸入名字：")
print("歡迎光臨，" + last_name + first_name + " 先生/小姐！")

```
