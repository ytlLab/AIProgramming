# Google Colab 簡介

Google Colaboratory（簡稱 Colab）是一款基於雲端的 Jupyter Notebook 環境。它最大的特色是完全免費且無需安裝，只要有 Google 帳號與瀏覽器即可撰寫並執行 Python 程式碼。

## 核心優點
- 免費 GPU/TPU 資源：這對機器學習與深度學習初學者極具吸引力，能大幅縮短運算時間。
- 雲端整合：程式碼自動儲存於 Google Drive，方便隨時切換裝置或與他人協作（類似 Google 文件的共享機制）。
- 預載環境：內建了常用的大數據與 AI 套件（如 NumPy, Pandas, TensorFlow），省去設定環境的煩惱。

# pyton 基礎語法

## 基本輸出輸入與變數
程式的基礎在於資料的流動：獲取資料並呈現結果。
- 變數賦值：x = 10
- 輸出：使用 print()。
- 輸入：使用 input() (注意：輸入進來的資料預設都是「字串」)。

### ex1：訪客歡迎程式
說明：請使用者輸入「姓氏」與「名字」，並將兩者合併，輸出：「歡迎光臨，[姓氏][名字] 先生/小姐！」
- 使用 + 號連結字串。
```
last_name = input("請輸入姓氏：")
first_name = input("請輸入名字：")
print("歡迎光臨，" + last_name + first_name + " 先生/小姐！")
```

### ex2：簡單計算機 (加法)
說明：請使用者輸入兩個整數，計算它們的和，並輸出：「計算結果為：[總和]」。
- 記得將 input() 轉換為 int()。
```
num1 = int(input("請輸入第一個整數："))
num2 = int(input("請輸入第二個整數："))
total = num1 + num2
print("計算結果為：", total)
```

### ex3：年齡計算器 (算術運算)
說明：請使用者輸入「出生年份」，程式自動計算到 2026 年時，使用者是多少歲，並輸出：「你在 2026 年時是 [歲數] 歲」。
```
birth_year = int(input("請輸入你的出生年份："))
age_in_2026 = 2026 - birth_year
print("你在 2026 年時是", age_in_2026, "歲")
```

### ex4：矩形周長計算 (括號優先權)
說明：請使用者輸入矩形的「長」與「寬」，計算周長（公式：$(長 + 寬) \times 2$），並輸出：「此矩形的周長為：[周長]」。
```
length = float(input("請輸入長度："))
width = float(input("請輸入寬度："))
perimeter = (length + width) * 2
print("此矩形的周長為：", perimeter)
```

### ex5：購物小幫手 (變數乘法)
說明：請使用者輸入「商品的單價」與「購買的數量」，計算總金額，並輸出：「總共需要支付：[金額] 元」。
```
price = int(input("請輸入單價："))
count = int(input("請輸入購買數量："))
total_cost = price * count
print("總共需要支付：", total_cost, "元")
```

### ex6：溫度轉換 (華氏轉攝氏)
說明：請使用者輸入華氏溫度，將其轉換為攝氏溫度（公式：$C = (F - 32) \times \frac{5}{9}$），並輸出：「目前的攝氏溫度為：[攝氏溫度]」。
```
f_temp = float(input("請輸入華氏溫度："))
c_temp = (f_temp - 32) * 5 / 9
print("目前的攝氏溫度為：", c_temp)
```

### ex7：圓周長計算 (浮點數運算)
說明：請使用者輸入圓形的「半徑」，計算圓周長（公式：$2 \times 3.14 \times 半徑$），並輸出：「圓周長等於：[周長]」。
```
radius = float(input("請輸入半徑："))
circumference = 2 * 3.14 * radius
print("圓周長等於：", circumference)
```

### ex8：三數平均值 (多變數處理)
說明：請使用者輸入三個數字，計算這三個數字的平均值，並輸出：「平均分數是：[平均值]」。
```
n1 = float(input("第一個數："))
n2 = float(input("第二個數："))
n3 = float(input("第三個數："))
avg = (n1 + n2 + n3) / 3
print("平均分數是：", avg)
```

### ex9：簡單找零系統 (變數減法)
說明：設定一個變數 money = 1000。請使用者輸入「消費金額」，計算剩餘的錢，並輸出：「找零金額：[餘額] 元」。
```
money = 1000
cost = int(input("請輸入消費金額："))
change = money - cost
print("找零金額：", change, "元")
```

### ex10：單位換算 (公分轉公尺)
說明：請使用者輸入身高（公分），將其換算為公尺，並輸出：「你的身高是 [公尺] 公尺」。
```
cm = float(input("請輸入身高(公分)："))
m = cm / 100
print("你的身高是", m, "公尺")
```

## 迴圈控制 Loop
重複執行重複的事情，是電腦最強大的功能。
- for 迴圈：通常用於已知次數或遍歷物件。
- while 迴圈：用於條件達成前持續執行的情境。

> range(起始, 結束, 步進)：注意「結束」是不包含在內的。例如 range(1, 5) 只會跑到 4。
> 
> break：當你想在條件達成時「立刻逃跑」出迴圈，它非常好用。
> 
> continue：跳過「這一次」後面的程式碼，直接進入下一次迴圈。

### ex1：基礎數字列印 (for)
說明：
```
for i in range(1, 11):
    print(i)
```

### ex2：指定次數的歡迎詞
```
n = int(input("請問要印幾次："))
for i in range(n):
    print("Hello Python")
```

### ex3：計算 1 到 N 的總和
說明：
```
n = int(input("請輸入一個正整數："))
total = 0
for i in range(1, n + 1):
    total = total + i
print("總和為：", total)
```

### ex4：偶數過濾器
說明：
```

```

### ex5：倒數計時器
說明：
```

```

### ex6：九九乘法表 (巢狀迴圈)
說明：

```
    for j in range(1, 10):
        print(i, "x", j, "=", i * j)
    print("---") # 分隔每一組數字
```

### ex7：猜數字遊戲 (while 迴圈)
說明：

```
        print("沒猜中，再試一次！")
```

### ex9：星號金字塔 (圖形輸出)

### ex10：累計輸入直到 0 為止
說明：
```

while True:
    num = int(input("請輸入數字 (輸入 0 結束)："))
    if num == 0:
        break # 強制跳出迴圈
    total = total + num
print("累計總和為：", total)
```


## 串列操作 List 
串列是 Python 中常用的資料結構之一，可以存放多個資料。

- 索引 (Index)：從 0 開始計算。
- 常用操作：append() 新增、len() 長度、切片 (Slicing)。

### ex1：建立與存取串列 (基礎)
說明：
```
fruits = ["蘋果", "香蕉", "橘子", "葡萄", "草莓"]
print("第一種水果是：", fruits[0])
print("最後一種水果是：", fruits[-1])
```

### ex2：修改串列內容
說明：
```
nums = [10, 20, 30]
nums[1] = 50
print("修改後的串列：", nums)
```

### ex3：新增與刪除元素
說明：
```
shop_list = []
item1 = input("請輸入想買的第一樣東西：")
shop_list.append(item1)
item2 = input("請輸入想買的第二樣東西：")
shop_list.append(item2)
item3 = input("請輸入想買的第三樣東西：")
shop_list.append(item3)

print("原始清單：", shop_list)
del shop_list[0]
print("刪除第一項後的清單：", shop_list)
```

### ex4：串列長度與加總
說明：
