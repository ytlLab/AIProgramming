# Pandas 資料清洗與探索

## 說明
Pandas 是 Python 生態系中最核心的資料分析套件，其名稱源自「面板資料」（Panel Data）。它提供了兩大主要資料結構：一維的 Series（類似有索引的陣列）與二維的 DataFrame（類似 Excel 表格）。Pandas 的強大之處在於其卓越的資料清洗與預處理能力，能輕鬆處理遺失值、重複資料及格式不一的數據，將混亂的原始資料轉化為可分析的結構化資訊。

支援多種檔案格式的讀寫（如 CSV、Excel、SQL、JSON），並具備強大的分組（GroupBy）、合併（Merge/Join）與重塑（Reshape）功能。透過向量化運算，它在處理大規模數據時效能遠優於傳統的 Python 迴圈。

核心資料結構 Pandas 主要由兩個核心組件組成：

- Series：一維的資料結構，類似於列表或數組，但帶有索引（Index）。
- DataFrame：最常用的二維表格結構。可以想像成一個 Excel 工作表 或 SQL 表格，由多個 Series（列）組成。

### 課程練習
> None 或 np.nan 代表缺失值

import pandas as pd

# 建立包含遺失值的模擬資料

3. 使用 .interpolate() 嘗試對數值型欄位的遺失值進行線性插補。

### 課程練習
建立一個包含日期、學生姓名與成績的 DataFrame
```
import pandas as pd
import numpy as np

# 建立 2024 年初的連續日期資料

    '學生姓名': [f'學生{i%5}' for i in range(60)],
    '成績': np.random.randint(50, 100, size=60)

df_ts = pd.DataFrame(data)

print("--- 原始時間序列資料 (前 5 筆) ---")

```
練習題：</br>

2. 撰寫一個自定義函數 classify_score(score)，將成績轉換為 A/B/C 等級，並用 .apply() 新增一欄。
3. 使用 .resample('M') 將日資料轉換為月資料，並計算每月平均值。
